# touch-bar

> Launch the macOS [Touch Bar simulator](https://github.com/sindresorhus/touch-bar-simulator) from the command-line


## Install

```
$ npm install --global touch-bar
```

*Requires macOS 10.14 or later.*


## Usage

```
$ touch-bar
```
